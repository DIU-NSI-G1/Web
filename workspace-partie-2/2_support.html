Objectifs globales

Le but de ce workspace est double :

  • Utiliser Javascript pour modifier le contenu HTML d'une page Web
  • Comprendre les différentes façons d'intégrer du style CSS et comprendre les effets du style sur une page HTML

Pour répondre à ces deux objectifs, le workspace est divisé en deux parties :

  • Modifier la structure HTML de la page 2_button_modify_head_content_page.html avec Javascript
  • Intégration et comprehension du style CSS appliqué sur la page 2_button_modify_head_content_page.html

Tâche 1

Analyser la page HTML

Ouvrez, à l'aide d'un navigateur, la page 2_button_modify_head_content_page.html et analysez son contenu.

Question 1

Que contient cette page Web?

Cette page web contient une page HTML composée d'un head et d'un body. Dans le head, il y a un script à modifier ainsi qu'un lien vers une feuille de style CSS. Dans le body, il y a un titre, un paragraphe, un bouton et un pied de page.

Question 2

Décrivez en quelques mots le style cette page Web.

Le style de cette page web définie la couleur et les bordures du titre principale de la page (h1), elle définie aussi la police utilisée dans le titre et dans le paragraphe. La dernière partie du style donne la valeur des marges des différents élements de la page HTML.

Dans l'état actuel, si vous essayez de cliquer sur le bouton Clique ici pour changer le titre de la page, rien ne se passe...

Question 3

Analysez la page HTML à l'aide d'un éditeur de texte. Pourquoi ne se passe-t-il rien lorsque vous cliquez sur ce bouton?

Dans le script de la page HTML il y a une fonction changeTitle(), mais elle ne contient que des commentaires, donc aucune instruction n'est effectuée.

Tâche 2

Compléter du code Javascript

Complétez la méthode changeTitle() de la page 2_button_modify_head_content_page.html.

Cette méthode doit changer le titre de la balise h1 identifier par l'attribut id="mainTitle" tel que décrit dans la documentation de la méthode.

Question

Après avoir complété cette méthode et rafraîchit la page, expliquez à présent ce qu'il se passe lorsque vous cliquez sur le bouton?

Lorsqu'on clique sur le bouton, la fonction changeTitle() est lancée et le titre change.

Tâche 5

Lire du CSS

Question 1

Expliquez l'effet des propriétés CSS de la page sur h1 2_button_modify_head_content_page.html.

Les différentes propriétés css de la page sur h1 sont :

  • "color" définit la couleur du texte ;
  • "background-color" définit l'interieur du cadre du titre ;
  • "border" définit les propriétés de la bordure épaisseur, couleur... ;
  • "padding" correspond à l'alinéa ;
  • "border-radius" donne son arrondi à la bordure de la zone de titre ;
  • "text-align" définit l'alignement du texte ;
  • "font-family" définit le type de police d'écriture.

Question 2

Expliquez l'effet des propriétés CSS de la page sur la balise <p>

Les différentes propriétés css de la page sur la balise <p> sont :

  • "font-family" définit le type de police d'écriture ;
  • "margin-left" crée une marge à gauche.

Question 3

Quelles balises HTML ont des propriétés CSS en communs? Justifiez.

Les balises HTML ayant des propriétes css en communs sont h1 et p ainsi que p, img et button.

Tâche 6

Intégration du CSS

Question 1

Expliquez comment le CSS a été séparé de la page Web 2_button_modify_head_content_page.html.

Le css a été séparé de la page Web en créant un fichier 2_button_modify_head_content_page.css appelé dans le head de la page HTML à l'aide de la balise "link".

Question 2

Comment feriez-vous pour intégrer le style CSS directement sur la page HTML?

Pour intégrer le style CSS directement sur la page HTML, on utilise la balise style type="text/css" inscrite à l'interieur du head.

Intégrer directement le style CSS dans le contenu de la page HTML

Question 3

Dans ce contexte, pensez-vous qu'il est préférable de mettre le style CSS directement dans le contenu de la page HTML ou qu'il vaut mieux que les 2 soient séparés? Justifiez.

Je pense qu'il vaut mieux que le CSS reste à part de la page HTML car cela rajoute beaucoup d'informations dans le head et il est plus difficile de s'y retrouver.
