Première partie - Modifier le style d'une page HTML existante à l'aide de Javascript

Dans cette première partie du workspace vous allez travailler sur partie javascript du fichier : 3_button_modify_css_head_page.html

En ce sens, vous allez devoir modifier le style HTML de cette page au fil des tâches à réaliser.

Cette première partie est complètement focus sur la partie Javascript.

A la fin de cette partie, vous serez interagir avec un bouton pour modifier le style avec Javascript

Tâche 1

Analyser la page HTML 3_button_modify_css_head_page.html

Question 1

Décrire en quelques mots le rendu et le contenu de cette page Web.

Il ne semble pas y avoir de rendus particuliers, les éléments s'affichent comme sur une page sans fichier css chargé. Contenu: un titre s'affiche dans l'onglet.On peut distinguer un h1, un paragraphe, un bouton et un lien. L'examen du fichier html confirme cela. Cependant, le lien se trouve en fait dans une balise footer.

Dans l'état actuel, si vous essayez de cliquer sur le bouton Clique ici pour changer le style CSS du titre de la page, rien ne se passe.

Question 2

Examinez la page à l'aide d'un éditeur de texte et expliquez le problème.

L'attribut événement onclick du bouton indique à la page web de réagir à l'événement de clic sur le bouton en lançant l'exécution de la fonction javascript changeTitleCSSStyle qui pour le moment ne fait rien.

Tâche 2

Compléter le code Javascript de la page 3_button_modify_css_head_page.html.

Complétez la méthode changeTitleCSSStyle() de la page

Cette méthode doit changer le style du titre de la balise h1 identifier par l'attribut id="mainTitle" telle que décrit dans la documentation de la méthode.

Deuxième partie - Du style avec ou sans l'aide de Javascript

Dans cette deuxième partie, le but est de différencier les manières de rajouter du style

Tâche 3

Style CSS avec Javascript ou sans Javascript pour la page 3_button_modify_css_head_page.html?

Question 1

Comment feriez-vous pour rajouter le même style CSS que vous avez codé dans la méthode changeTitleCSSStyle() avec Javascript mais cette fois-ci dès le chargement de la page, sans passer avec un bouton?

Dans la balise script précédente, on peut réaliser l'affectation suivante: window.onload = changeTitleCSSStyle;

Question 2

Rajoutez le même style CSS sans passer par Javascript. Expliquez les différences entre les deux méthodes.

On peut ajouter directement un attribut style dans la balise h1: style = " color : red; background-color : yellow; border: thick double #32a1ce; Dans ce nouveau cas, le style est interprété directement pendant la construction de la page web par le navigateur. Avec l'utilisation de l'affectation de window.onload le style est appliqué seulement après l'interprétation du reste de la page.

Normalement, si vous avez rajouté le même style, plus rien ne devrait se passer lorsque vous cliquez sur le bouton car le style que vous essayez d'appliquer est exactement le même que celui de la page

Troisième partie - Appliquer du style à l'aide de Javascript

Dans cette troisième partie, le but est de ré-utiliser du code javascript existant et de le modifier

Tâche 4

S'inspirer du code Javascript existant de la page 3_button_modify_css_head_page.html

Rajoutez au minimum une méthode Javascript qui modifie le style CSS du bouton Clique ici pour changer le style CSS du titre de la page. Ces méthodes doivent être appelées en même temps que la méthode changeTitleCSSStyle() lorsqu'on appuie sur ce même bouton.

Le nombre minimum de style que vous devez rajouter à ce bouton est 9.

Tâche 5

S'inspirer du code Javascript existant la page 3_button_modify_css_head_page.html

Choisissez au moins 9 style CSS qui se différencient de tous les autres précédents et rajoutez les au paragraphe de la page Ceci est un exemple d'intéractivité entre Javascript et le contenu HTML d'un document seulement lorsque l'utilisateur déplace le curseur de la souris sur lui.

Tâche 6

Ré-utiliser du code Javascript

A l'aide de javascript, changer de manière aléatoire (random) la couleur de l'arrière-plan dès qu'on appuie sur le bouton "Clique ici pour changer l'arrière-plan"
