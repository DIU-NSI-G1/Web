Objectif global

Plusieurs personnes ont soumis des idées pour le projet. Je vous propose donc, d'implémenter en HTML, CSS, Javascript ce que vous aviez en tête. Toutefois, il faudra trouver le moyen d'utiliser toutes les notions vu dans ce workspace dans votre idée.

Pour ceux qui n'ont pas d'idée, je vous propose de vous amusez avec les objects en Javascript dans ce workspace.

Tâche 1

Créez des objets simples

En vous inspirant de l'objet dog, créez 3 autres objets :

  • cat avec les propriétés et valeurs suivantes :
    • sound= Miaow
    • name= Mittens
  • pig avec les propriétés et valeurs suivantes :
    • sound= Oink
    • name= Charlie
  • horse avec les propriétés et valeurs suivantes :
    • sound= Neigh
    • name= Marie

Question

Comment feriez-vous pour qu'il partage tous les trois la méthode speak()?

Il me semble que les trois objets cat, pig et horse peuvent utiliser la méthode speak() car ils ont tous les trois les élements sound et name qui sont utilisés dans speak().
On ajoute alors un argument que nous appelons "animal" à l'interieur des parenthèses puis nous remplaçons les "this" par "animal" dans la dernière ligne.

Edit Thomas Gire: On peut aussi ajouter directement la méthodes speak aux trois animaux dog.speak = speak et laisser la référence à l'objet this.

Modifiez vos 3 objets ainsi que l'objet dog en conséquence.

Tâche 2

Reutiliser du code

Réutilisez la fonction appendIn() pour afficher sur la page HTML le résultat de la méthode speak() sur les 3 objets que vous avez créé

Tâche 3

Creation d'un constructeur Animal

Définir un constructeur Animal pour créer des animaux avec les mêmes propriétés que le constructeur Car et avec les mêmes propriétés que l'objet dog

Rajouter une dernière propriété pour définir la race de l'animal

Tâche 4

Apeler le constructeur Animal

Utilisez le constructeur Animal pour créer trois animaux avec des valeurs aux choix :

  • Chien
  • Chat
  • Cheval

Utilisez les images fournies pour représenter chacun d'eux sur la page HTML

Tâche 5

Dessiner vos Animaux

Inspirez vous des méthodes associées à Car pour dessiner sur la page vos trois animaux.

Chacun de vos animaux doit apparaître distinctement sur la page HTML.

Tâche 6

Déplacer vos Animaux

Inspirez vous des méthodes associées à Car pour déplacer sur la page vos trois animaux.

Déplacer vos trois animaux près des rebord de la page HTML.

Edit Thomas Gire: On a choisi de faire apparaitre nos trois animaux initialement aux trois bords de la fenêtre du navigateur, en utilisant les propriétés de la fenêtre window.innerHeight et window.innerWidth.

Chacun de vos animaux doit apparaître distinctement sur la page HTML.

Tâche 7

Challenge : Déplacer un de vos animaux avec les quatres flèches du clavier

Ajoutez les évènements claviers correspondants aux flèches du clavier à votre Animal de type chat.

Exemple : Si c'est la flèche du bas qui est appuyée alors le chat se déplace vers le bas. Si c'est la flèche qui est appuyée de droite alors le chat se déplace vers la droite...
