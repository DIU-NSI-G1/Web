Tâche 1

Interaction avec la page HTML

Examinez la page HTML Js_can_change_the_table_content.html dans le navigateur.

Question 1

Qu'affiche cette page?

Réponse

Cette page affiche un titre dans le head, un titre h2 et un bouton dont la fonction est de construire un tableau. Puis, elle affiche les headers Nom et Prénom d'un tabeau.
Enfin, cette page contient un footer qui possède un lien vers la page 6_support.html.

Question 2

Quelles sont les moyens pour l'utilisateur d'intéragir avec cette page?

Réponse

Javascript et CSS sont les deux principaux moyens d'agir avec la page courante par l'intermédiaire du bouton (par exemple). Dans le cas qui nous concerne, c'est Javascript qui permettra à l'utilisateur d'interagir avec la page.

Tâche 2

Examiner la page HTML Js_can_change_the_table_content.html à l'aide de votre éditeur

Dans l'état actuel, si vous essayez de cliquer sur le bouton, rien ne se passe

Question

Ouvrez la page à l'aide d'un éditeur de texte. Quelle est le problème?

Réponse

Ni le bouton, ni aucun autre objet n'a d'effet visible sur la page. Si on regade le fichier Js_can_change_the_table_content.js, on remarque que la fonction addLineToHTMLTable n'est pas implémentée. C'est pour cette raison que le bouton n'a aucun effet.

Tâche 3

Compléter du code javascript

Complétez la fonction addLineToHTMLTable() du fichier Js_can_change_the_table_content.js pour ajouter une ligne avec deux cellules; une contenant le prénom et l'autre contenant le nom

Indice

Regardez les liens de la section "Pour vous aider..."

Question

Après avoir complété cette fonction, essayez de cliquer de nouveau plusieurs fois sur le bouton. Quelles sont les modifications apportées sur la page?

Réponse

Après chaque clique sur le bouton 3 lignes sont ajoutées à la fin du tableau. Chaque groupe de 3 lignes ajoutées contient les trois mêmes données que le groupe précédent.
C'est à dire, à chaque clique on ajoute (dans l'ordre):

  • "Alice", "Belle"
  • "Bob", "Gentil"
  • "Melchior", "Mechant"

Tâche 4

Apporter de nouvelles fonctionnalités

Examinez la fonction buildTable() du fichier Js_can_change_the_table_content.js

Question

Que fait cette fonction?

Réponse

La fonction buildTable() est appelée lors de la survenue d'un événement onclick sur le bouton considéré.
Elle a pour action d'appeler la fonction addLineToHTMLTable(fName, lName); dont le rôle est d'afficher une nouvelle ligne.

Modifier la page HTML et code javascript pour rajouter 2 inputs : prénom et nom

Lorsque l'utilisateur appuyera sur le bouton, c'est le prénom et le nom que l'utilisateur aura indiqué dans les input qui seront insérées dans le tableau et non des valeurs en dures.
